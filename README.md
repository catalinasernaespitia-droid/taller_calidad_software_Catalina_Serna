El proyecto Ferretería es un sistema web diseñado para gestionar de manera eficiente las operaciones principales de una tienda de ferretería.
Permite administrar productos, categorías, usuarios y procesos internos, facilitando el control del inventario y el acceso seguro al sistema.

Este sistema busca optimizar el flujo de trabajo dentro de la ferretería, ofreciendo una interfaz sencilla, rápida y fácil de usar para el administrador o encargado.
Incluye funciones esenciales como gestión de artículos, control de stock, inicio de sesión seguro y un panel principal para visualizar la información más importante del negocio.

Tecnologías utilizadas

PHP – lenguaje principal para la lógica del sistema.

MySQL / MariaDB – base de datos para almacenar productos, usuarios, categorías y demás información del sistema.

XAMPP – servidor local para desarrollar y ejecutar la aplicación.

Otras herramientas

Git y GitHub – control de versiones.

phpMyAdmin – administración visual de la base de datos.

Hash de contraseñas (password_hash) – seguridad para los usuarios.
